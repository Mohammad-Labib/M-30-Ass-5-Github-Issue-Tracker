5️⃣ What are template literals?
=> Template literals are JavaScript string literals, introduced in ES6, that allow for embedded expressions (${expression}), multi-line strings, and string interpolation. Enclosed by backticks (`) instead of quotes, they improve readability and eliminate the need for complex string concatenation.
